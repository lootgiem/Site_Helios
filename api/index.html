To reproduce the issue: use a small screen width (about 600px) so the hamburguer menu icon appears at the left.

Then click the icon.

Two things should happen:

  1. The side nav should open
  2. An alert should appear

However, the alert is not being shown.

The reason is that the sideNav plugin is preventing event bubbling, what makes any other event handlers on the toggler element that are delegated not to trigger.

To double check, just comment the line $(".button-collapse").sideNav(); and run again. In doing that, the other handler (with the alert) will trigger just fine.
